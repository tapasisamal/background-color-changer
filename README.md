# Background Color Changer

A simple web application that changes the background color when you click on colored blocks. Built using HTML, CSS and JavaScript.

## Features
- Click any color block to change the page background color
- Instant and smooth color switching
- Simple, clean, and interactive UI

## Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

## What I Learned
- Handling click events in JavaScript
- Using DOM to dynamically update styles
- Creating interactive UI with basic JavaScript
